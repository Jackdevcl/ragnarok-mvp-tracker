# 👑 Ragnarok Online - MVP Tracker Premium

Proyecto open-source para la comunidad de RO, diseñado para que parties y guilds sincronicen tiempos de respawn con precisión milimétrica.

## ✨ Características

- **Smart Sorting**: Ordenamiento inteligente de MVPs para optimizar tus rutas.
- **Alarmas de Audio**: Alertas de sonido precisas para no perder ningún spawn.
- **Party Sync en tiempo real**: Sincronización instantánea entre todos los miembros de la party.
- **Temas Claro/Oscuro**: Interfaz adaptable a tus preferencias visuales.

## 🛠️ Tecnologías

Este proyecto está construido con un stack moderno:
- **Frontend**: Angular (Standalone Components, Signals)
- **Backend**: Node.js con Express
- **Base de Datos**: PostgreSQL / MySQL
